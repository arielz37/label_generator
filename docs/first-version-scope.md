# 第一版演示范围

## 目标

第一版用于 presentation，证明标签自动生成器的核心流程可行：

1. 输入批次信息
2. 生成标签
3. 展示标签
4. 一键打印

这一版暂时不连接 ERP，不真实调用 BarTender，不处理全部历史模板。演示数据只覆盖 5 个已经手动标准化的客户模板。

## 核心假设

- 客户配置中提前维护“是否使用客户指定标签”。
- 客户指定标签只处理已接入自动化的模板。
- 每个已接入模板使用统一字段名。
- 每一张标签的数据由系统生成，BarTender 模板只负责排版、条码、二维码渲染。
- 序列号、箱号、罐号等可变信息按每张标签生成一行数据。

| 客户编号 | 客户名称 | 标签来源 | 模板命名方式 |
|---|---|---|---|
| CUST-ALPHA | Alpha Semiconductor | 客户指定标签 | 客户货号 + 内标/外标 |
| CUST-BETA | Beta Micro | 客户指定标签 | 我司货号 + 内标/外标 |
| CUST-GAMMA | Gamma Devices | 客户指定标签 | 客户货号 + 内标/外标 |

## 后续真实版本需要补齐

- BarTender 命令行调用方式确认
- `.btw` 模板字段绑定规范
- PDF 或图片预览生成方式
- 真实打印机 `TSC TTP-244Pro` 打印验证
- 客户配置持久化
- 打印日志持久化
- ERP 数据库读取
